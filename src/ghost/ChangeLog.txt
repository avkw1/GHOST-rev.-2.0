1.0.0 - 17.06.2013 - Первая версия.
1.1.0 - 15.09.2013 - Добавлены макросы для платы rev. 2.0. Изменён алгоритм вы-
                     дачи (первой всегда отправляется карта). По умолчанию PIN-
                     коды из одной цифры.
1.2.0 - 27.01.2014 - Заменён загрузчик (добавлена защита от чтения flash).
                     Добавлен адрес прибора и работа с RS-485 (переход в режим
                     загрузчика). Добавлены биты "Норма" и "Алкоголь" в маске выдавае-
                     мых событий алкотестера.
1.3.0 - 19.02.2014 - Добавлен параметр "Запоминать только первую карту".
1.4.0 - 09.09.2014 - Добавлена возможность выключения отправки карт, таблица
                     кодов клавиатуры, возможность выключения PIN-кодов.
                     Увеличена пауза между картами по-умолчанию (2000 мс).
1.4.1 - 07.10.2014 - Добавлена выдача кода "Готов" от алкотестера (без настроек).
1.5.0 - 27.10.2014 - Добавлен алгоритм 2 (карта перед выходом) с настройками.
                     Различные оптимизации в разных модулях. Светодиод мигает 1 раз
                     для каждого отправленного кода карты.
1.6.0 - 25.02.2015 - Добавлен параметр "Отправка показаний без кода карты" (только
                     для алгоритма 2 ).
1.6.1 - 15.04.2015 - Исправлен баг в алгоритме 2 при выдохе без карты.
1.7.0 - 01.12.2015 - Полностью реализован протокол обмена по RS-485. Длина команд
                     уменьшена, теряется совместимость со старыми версиями ПО.
1.7.1 - 03.02.2016 - Обратный порядок отправки карт для алгоритма 2 (сначала пока-
                     зания алкотестера, затем карта со считывателя).
1.7.2 - 15.02.2016 - Отменён обратный порядок карт. Пауза после PIN-кода разделена
                     на 2 настраиваемых параметра: пауза между посылками и пауза
                     после отправки ENTER.
1.8.0 - 24.01.2017 - Исправлено пропадание связи по RS-485 во время обработки карт,
                     добавлена команда подтверждения получения данных. Изменился
                     код команды управления реле (теряется обратная совместимость
                     протокола).
1.9.0 - 14.02.2018 - Поддержка шифрованных данных от алкотестера.
