"0" - бит запрограммирован
"1" - бит не запрограммирован

FUSE биты
---------

SELFPRGEN = 0       ; Разрешить самопрограммирование flash
DWEN      = 1
EESAVE    = 1
SPIEN     = 0       ; [!] Разрешить последовательное программирование
WDTON     = 1

BODLEVEL2 = 1       ; BOD включён, сброс при VCC = 4,3 В
BODLEVEL1 = 0
BODLEVEL0 = 0

RSTDISBL  = 1
CKDIV8    = 1
CKOUT     = 1

SUT0      = 0       ; Внешний кварц 4 МГц, BOD включён
SUT1      = 1
CKSEL3    = 1
CKSEL2    = 1
CKSEL1    = 0
CKSEL0    = 1

EXTENDED  = 0xFE    ; В виде байтов
HIGH      = 0xD9
LOW       = 0xDD


Биты защиты (Lock bits)
-----------------------

LB1       = 0       ; Запрет программирования и чтения памяти из контроллера
LB2       = 0

LOCKBIT   = 0xFC    ; В виде байта
